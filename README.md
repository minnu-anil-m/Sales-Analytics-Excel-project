# Sales-Analytics-Excel-project
Excel-based sales data analysis project using the Ferns &amp; Petals dataset to derive insights on revenue, customer spending, and order patterns.

Creating an excel based dashboard using a dataset from FNP (Ferns and Petals) that specializes in sending gifts for
various occasions like Diwali, Raksha Bandhan, Holi, Valentine's Day, Birthdays, and
Anniversaries. The dataset contains details about the products, orders, customers, and relevant
dates.The task is to analyze this dataset to uncover key insights related to sales trends,
customer behavior, and product performance.
